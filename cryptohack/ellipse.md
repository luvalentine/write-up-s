# Ellipse curve cryptography

## Files
- `source.py` - скрипт, который генерирует/проверяет значения по формулам
- `output.txt` - готовый вывод скрипта

## Idea
В `source.py` логика построена вокруг конических сечений (conic sections): круг/эллипс/парабола/гипербола.  
Скрипт перебирает набор случаев и для каждого:
1) считает значения по формулам  
2) проверяет условия (валидность случая)  
3) если условие проходит - добавляет соответствующий символ в итоговую строку/печатает его

То есть флаг формируется не “шифрованием”, а как результат прохождения проверок.

## Solution
`output.txt` уже содержит правильный результат работы скрипта - последовательность символов, которые прошли все условия.  
Достаточно собрать символы в порядке вывода и обернуть в формат `crypto{...}`.
